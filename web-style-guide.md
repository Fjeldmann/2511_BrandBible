# Fjeldmann – Web Styleguide

## 1. Brand Essence (for context)

Fjeldmann builds **human-centered, simple, and enjoyable software**.
Keywords: *menneskelig, grundig, konstruktiv*.

---

## 2. Typography

### Primary Typeface — **Inter**

Used for all headings, paragraphs, UI labels, and long-form text.

| Element   | Font  | Weight  | Size (web guideline) | Notes                                     |
| --------- | ----- | ------- | -------------------- | ----------------------------------------- |
| H1        | Inter | Bold    | 3.5–4rem             | For main page titles                      |
| H2        | Inter | Bold    | 2.5–3rem             | Section titles                            |
| H3        | Inter | Bold    | 2–2.25rem            | Subsections                               |
| H4        | Inter | Bold    | 1.5–1.75rem          | Smaller headers                           |
| H5        | Inter | Bold    | 1.25rem              |                                           |
| H6        | Inter | Bold    | 1rem                 |                                           |
| Paragraph | Inter | Regular | 1rem                 | Line height 150%                          |
| Links     | Inter | Regular | 1rem                 | Underline on hover, colour change allowed |

### Secondary Typeface — **Montserrat**

Use for:

* Buttons
* Labels
* UI microtext requiring emphasis

**Style:**

* Bold
* All caps
* +10–20% letter spacing (slight tracking)

## 3. Colour Palette

### Core Brand Colours

| Name          | Hex       | Usage                            |
| ------------- | --------- | -------------------------------- |
| **Sort blæk** | `#222222` | Primary text, dark UI elements   |
| **Mørkegrå**  | `#6D6D6D` | Secondary text                   |
| **Mellemgrå** | `#D5D5D5` | Borders, dividers, lines         |
| **Lysegrå**   | `#F8F8F8` | Backgrounds, cards, sections     |
| **Hvid sne**  | `#FFFFFF` | Main background / negative space |

### Accent Colours

| Name            | Hex       | Usage                                       |
| --------------- | --------- | ------------------------------------------- |
| **Gyldent rav** | `#DC851F` | Primary accent, highlights, key UI elements |
| **Varmt sand**  | `#E0BB89` | Soft accent backgrounds                     |
| **Blågran**     | `#025951` | Secondary accent (sparingly)                |

### Colour Usage Guidelines

* Prefer **soft neutrals** and **high clarity** layouts.
* Use accents only to guide attention (CTAs, key interactions).
* Avoid placing the **orange logo** on **greenish backgrounds**.
* Maintain strong contrast for accessibility.

---

## 4. Logo Usage

### General Rules

* Do **not** use the mark alone unless the “Fjeldmann” name is nearby.
* Maintain **minimum spacing equal to the width of the “F”** surrounding the logo.
* Minimum sizes:

  * **Logo mark:** 16px width
  * **Logotype:** 60px width

### Colour Rules

**Use:**

* Coloured logo on light, calm backgrounds
* White logo on dark or colourful backgrounds
* Black logo in grayscale contexts or next to other brand logos

**Avoid:**

* Coloured logo on busy or colour-heavy backgrounds
* White logo on light backgrounds
* Black logo on dark backgrounds
* Altering proportions, colours, rotation, outlines, or fill effects

---

## 5. Buttons & UI Elements

### Primary Buttons

* Font: **Montserrat Bold**
* All caps
* 16pt / ~1rem
* +10% tracking
* Use **Gyldent rav (#DC851F)** as primary colour
* Maintain generous padding and rounded corners (Fjeldmann = soft values)

### Secondary Buttons

* Montserrat Bold, all caps
* Neutral greys
* Focus on clarity over decoration

### Labels / Tags

* Montserrat Bold
* All caps
* Slight tracking

---

## 6. Imagery

### Photography Style

* Human-centric: people × tech × nature
* Deep tones, soft backgrounds (shallow depth of field)
* Natural light, authentic expressions
* Avoid sterile corporate stock photos

### Compositional Feel

* Calm, focused, optimistic
* Visual balance with negative space for text overlays
* For busy images, apply a **50% white overlay** before using coloured logo

---

## 7. Textures (optional enhancement)

Used sparingly in backgrounds or hero sections:

* Rope
* Stone
* Soft-focus surfaces
* Skin
* Bark

Keep subtle and non-intrusive.

---

## 8. Icons & Graphics

* Use **outlined Material Symbols** (Google)
* Clean, geometric, minimal
* Communicate simplicity & reduced complexity
* Consistent stroke weight

---

## 9. Tone of Voice (for web copy)

Short, warm, constructive, human.

* **Address the reader as “du”.**
* **Optimistic**, but not naive.
* Use examples to explain technical concepts.
* Use humour lightly: *kæk, ikke fræk*.
* Emphasise action, clarity, and improvement.
* “Glasset er halv fyldt.”

**Tagline:**
*Skarp software med bløde værdier.*

**Short CTA:**
*Mærk forskellen med menneskelig software.*
